# Liverpool Managers

Graphy stuff

## Prerequisites

- ruby 2.4.0
- node 11

## initialise

```bash
cd ./packages/api
docker-compose up -d
cd ...
ruby ./build.rb
```

## Clean up

```bash
docker-compose rm -f -v
```
